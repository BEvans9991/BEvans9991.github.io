Inverse Kinematics & Procedural Animation
This project was made to explore inverse kinematics and procedural animation within games. This was implemented using the FABRIK (Foward-and-Backward Reaching Inverse Kinematics) algorithm.

Repository
Destructable Asteroids
This project procedurally generates asteroids that you can destroy! The underlying technique for the generation and destruction of these asteroids is the Marching Squares algorithm. The asteroids are distributed in space using a Poisson Disk Sampling algorithm.
